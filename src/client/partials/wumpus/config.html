Key Config

←
Discrete: Turn the agent left.
Continuous: Apply force to increase agent count-clockwise velocity.
→
Discrete: Turn the agent right.
Continuous: Apply force to increase agent clockwise velocity.
↑
Discrete: Move the agent forward.
Continuous: Apply force to increase agent velocity.
↓
Continuous Only: Apply force to increase agent velocity.
G
Grab the gold, if the agent is in the correct room.
E
Exit the cave. The agent must have the gold and be in the correct room.
F
Multi Only: Fire an arrow. It will fly straight until it either hits a wall or slays the wumpus.
(Spacebar)
Static Only: Take no action for a round.
